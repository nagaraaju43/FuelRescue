# Location-Based Emergency Fuel Request System

A full-stack web application that helps users request emergency fuel assistance by locating nearby petrol pumps using live map-based discovery.

## Features
- User authentication (Login & Register)
- User dashboard with emergency-focused UI
- Live nearby petrol pump discovery (map-based)
- Route & ETA estimation
- Emergency fuel request flow
- Admin-side management (planned)

## Tech Stack
- Frontend: React, Tailwind CSS, Leaflet
- Backend: Node.js, Express (in progress)
- Maps: OpenStreetMap, Overpass API
- Routing: OSRM

## Project Status
Frontend (User flow) completed  
Backend & Admin modules in progress

## Disclaimer
This project is a prototype built for learning and demonstration purposes.
